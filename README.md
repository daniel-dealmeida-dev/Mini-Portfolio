## Descrição

Este projeto é um **portfólio moderno e responsivo**, construído com **HTML5 e CSS3**, projetado para mostrar informações pessoais, links para GitHub e LinkedIn, e uma foto de perfil.  

O design utiliza:
- **Gradientes de fundo** para destaque visual;
- **Efeitos de hover** no card para interatividade;
- **Layout responsivo** para se adaptar a diferentes tamanhos de tela;
- **Fonte Roboto** importada do Google Fonts.

O objetivo é servir como um **mini portfólio online**, ideal para apresentar suas informações de forma elegante e profissional.


## Tecnologias Utilizadas

- HTML5  
- CSS3  
- Google Fonts (Roboto)  
- Gradientes e box-shadow para design moderno  
- Flexbox para centralização e responsividade  


## Como Usar

1. Clone o repositório:

bash
git clone https://github.com/daniel-dealmeida-dev/portfolio-card.git
Abra o arquivo index.html no navegador.

Para personalizar:

Substitua a foto EU.jpg pela sua foto de perfil.

Altere o nome e descrição no HTML.

Atualize os links do GitHub e LinkedIn.

Recursos e Funcionalidades
Card interativo com efeito de levitação ao passar o mouse.

Botões com hover animado.

Totalmente responsivo para mobile e desktop.

Estrutura limpa e fácil de personalizar.
